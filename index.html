stuff I do

Services

Frontend Development
Portfolios, landing pages and dashboards built with clean HTML, CSS and JS. I handle everything from design to deployment on GitHub Pages or wherever you need it.
Backend Development
Server-side logic, APIs, databases and authentication for sites that need to do more than just look good. I set up Firebase, Node.js or whatever fits the project and connect it cleanly to your frontend.
FiveM Client Development
Custom FiveM clients built for real performance, clean menus, reliable feature sets and solid anti-detection. Built and tested properly, not thrown together.
Roblox Client Development
Full Roblox clients like ESP, aimbot, pathfinding AI and HWID-protected loaders. Clean code with solid anti-detection. I know what I'm doing here.
CS2 Client Development
Custom CS2 clients with clean menus, reliable feature sets and solid anti-detection. Built and tested properly for Valve's engine.
Call of Duty Mobile Client Development
Custom Call of Duty Mobile clients built for real performance, clean UI and reliable anti-detection on mobile.
Database & API Integration
Firebase, REST APIs and real-time data syncing between your bot, dashboard and website. I structure it so everything talks to each other without breaking.
Payment & Subscription Integration
Stripe and PayPal setup for subscription tiers, one-time payments and automated billing. Connected directly to your dashboard so payments and access stay in sync.
Key & License Systems
Key generation, HWID binding, expiry, tiers and revocation. Everything synced between your client, bot or panel in real time.
Discord Bot Development
Occasional Discord bot builds. Slash commands, HWID verification, subscription tiers, Firebase sync and auto-delivery, built on Discord.js v14 when a project calls for it.
