# Security Policy

## Found a vulnerability?

If you spot a security issue with this project, please don't post it publicly as an issue. Just email me directly instead.

**Contact:** jhnergarcia@gmail.com

When you reach out, it helps if you can include:
- What the issue is
- How to reproduce it
- What kind of impact it could have

## What to expect

I'll try to respond and look into valid reports as soon as I can. This is a personal portfolio project I maintain on my own, so response time depends on how busy things get, but I do read and take every report seriously.

## What this covers

This only applies to the code and content in this repo. If the issue is actually with a third-party service I use (like Formspree or Google Fonts), that's on them to fix, so report it to their team directly instead.
